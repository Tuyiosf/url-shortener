## 🚀 Running the Application

url-shortener is a backend service. That means it runs on your computer or a server and manages short URLs behind the scenes.

### Starting the service

Still inside the url-shortener folder in your terminal, type:

```
npm start
```

This command will start the application. You should see messages confirming it is running and listening for links.

### How to use it

Once the app is running:

- Open your web browser.
- Go to the local address, usually:

  ```
  http://localhost:3000
  ```

- You will see options for entering a long URL and getting a short one.
- You can also view stats and manage expiry dates for your links.
